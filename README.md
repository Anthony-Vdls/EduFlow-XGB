# EduFlow-XGB
---
This is a project done for MSU Denver's machine learning class.  
It is a streamlit hosted app. It contains a XGB model. 

## Run the app
* Option 1: Visit this [website](https://xgb-career-predictor.streamlit.app/) to interact with the app.
* Option 2: Run locally. After cloning the repo run from repo root
    ```
    python3 -m venv .venv
    source .venv/bin/activate
    pip install -r requirements.txt
    streamlit Welcome.py
    ```
